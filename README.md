### Future Gamers
*Bienvenidos a mi proyecto de React Js, creado por Alejo Veron.*

El proyecto se arranca con la aplicacion React Create App, utilizando la libreria de React Js.

La app se trata de un E-commerce el cual en su pagina principal cuenta con una barra de navegacion, con la cual se podran filtrar los productos por categoria. En su cuerpo principal la pagina muestra los articulos y mediante un boton se podra ver su detalle y posteriormente seleccionar la cantidad que deseemos comprar para enviarla al carro de compras, en el cual se podra finalizar el proceso de compra.
